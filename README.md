# Hola! Instrucciones

Luego de hacer fetch debes:
1. Crear un Python Environment y correrlo.
2. Correr `pip install -r requirements.txt`.
3. Luego correr `py manage.py runserver`
4. En el navegador web, ve a la url [localhost:8000](http://localhost:8000)

## Para descargar los datos en CSV:
- Ir a la url [localhost:8000/exportar](http://localhost:8000/exportar)

Saludos!
